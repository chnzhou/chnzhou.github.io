## 如何使用github发布静态网站

2022年11月2日



步骤：

1. 首先得有个github的账号
2. 还要有一个用来保存静态网站代码的仓库
3. 在仓库中使用github pages进行发布

注意事项：

1. 仓库得是公开的，否则不能免费使用github pages
2. 仓库得具有“username.github.io”格式的名字，否则通过username.github.io不能访问到你的网站
